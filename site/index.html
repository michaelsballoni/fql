My Media Search - Find and enjoy media files on your PC

Get search results for pictures, music, and videos in mere seconds
  • Search using keywords like "vacation videos -Michael" for all vacation videos that don't feature Michael

  • Tracks changes to folders that you've told it to look in while it is running

  • Open, Show in Folder, and Get Properties of search results

  • Switch search results between details and tiles

Download installer for Windows 10: MyMediaSearch.1.2.0.msi
